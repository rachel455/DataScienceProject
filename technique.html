Techniques used for this Project

The techniques I used were mostly from pandas and matplotlib.pyplot libraries. I read the csv files using pd.read_csv(). Then I used the .replace() and pd.to_numeric() method to change values of a column from a string to an integer. I modified the orginal dataset to make a new one to separate the columns that I wanted from the ones I did not need using .groupby(), .aggregate(), and .unstack(). Finally, I ploted the data using .plot() and other matplotlib.pyplot functions to customize and add more detail to the graphs and saved them as .png files.
